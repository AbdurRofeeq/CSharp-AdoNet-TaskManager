# C# ADO.NET Task Manager

A beginner-friendly console application demonstrating ADO.NET fundamentals with MySQL. Perfect for learning raw database operations in C# without ORMs.

## What You'll Learn

- **ADO.NET Core Concepts**: Connections, Commands, DataReaders
- **MySQL Integration**: Real database operations with MySQL
- **CRUD Operations**: Create, Read, Update, Delete tasks
- **Parameterized Queries**: Security best practices to prevent SQL injection
- **Database Design**: Proper table structure and relationships
- **Error Handling**: Robust database operation handling

## Tech Stack

- **C#** (.NET 6/7/8)
- **ADO.NET** (System.Data)
- **MySQL** Database
- **MySql.Data** NuGet Package

## Features

- Add, view, update, and delete tasks
- Mark tasks as complete/incomplete
- View tasks by status (pending, completed, overdue)
- Task statistics and due date tracking
- Secure parameterized queries
- Proper connection management

## Perfect For

- C# developers learning database fundamentals
- Understanding what happens behind ORMs like Entity Framework
- Beginners transitioning from in-memory data to persistent storage
